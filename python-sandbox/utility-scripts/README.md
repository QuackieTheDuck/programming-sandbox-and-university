# Python Utility Scripts

A collection of practical script tools developed to automate networking tasks and data manipulation. These utilities serve as practical helper tools for engineering and cybersecurity workflows.

### 🛠️ Script Index

`ascii_text.py`:
-converts ascii code to a char

`Base64_txt.py`:
-decodes base64 string

`Base64.py`:
-encodes hex string in base64

`down_detector.py`:
-pings provided web page to determine wheater page is up or not

`Hex_To_ASCII.py`:
-convert hex values to ascii then prints it

`Hex_txt.py`:
-allow user to enter hex values to convert them to ascii

`Long_Int_ToASCII.py`:
-converts long int into ascii values

`Num_To_ASCII.py`:
-convert decimal numbers to ascii chars
